# Алгоритм Евклида

**Алгоритм Евклида** используется для нахождения **наибольшего общего делителя** (НОД) двух целых чисел.

> **НОД** — это наибольшее целое число, на которое оба исходных числа делятся без остатка.

**Например:**

НОД(48, 18) = 6  
НОД(7, 3) = 1  
НОД(10, 10) = 10  

#### Алгоритм основан на следующем важном свойстве:

> НОД двух чисел не меняется, если большее число заменить на остаток от деления большего на меньшее.

`gcd(a, b) = gcd(b, a % b)`

Это позволяет последовательно уменьшать числа, пока одно из них не станет равно 0.

---

## Алгоритм с делением

1. Большее число делим на меньшее.
2. Если делится без остатка, то меньшее число и есть НОД (следует выйти из цикла).
3. Если есть остаток, то большее число заменяем на остаток от деления.
4. Повторяем шаги, пока остаток не станет равен 0.

### Пример
[Код алгоритма с делением](./gcd-division.js)
```javascript
function gcd(a, b) {
  while (b !== 0) {
    [a, b] = [b, a % b];
  }
  return Math.abs(a);
}

console.log(gcd(48, 18)); // 6
console.log(gcd(101, 103)); // 1
console.log(gcd(56, 98)); // 14
console.log(gcd(-24, 36)); // 12,
```
> `Math.abs` используется, чтобы корректно работать с отрицательными числами.

```
НОД(48, 18):
48 % 18 = 12
18 % 12 = 6
12 % 6  = 0
```
Когда остаток стал `0`, делитель (`6`) — это **НОД**.

---

## Алгоритм с вычитанием

1. Из большего числа вычитаем меньшее.
2. Если получается 0, то значит, что числа равны друг другу и являются НОД (следует выйти из цикла).
3. Если результат вычитания не равен 0, то большее число заменяем на результат вычитания.
4. Переходим к пункту 1.

### Пример
[Код алгоритма с вычитанием](./gcd-subtraction.js)
```javascript
function gcd(a, b) {
  a = Math.abs(a);
  b = Math.abs(b);

  while (a !== b) {
    if (a > b) {
      a -= b;
    } else {
      b -= a;
    }
  }
  return a;
}

console.log(gcd(48, 18)); // 6
console.log(gcd(101, 103)); // 1
console.log(gcd(56, 98)); // 14
console.log(gcd(-24, 36)); // 12,
```

```
НОД(48, 18):

48 - 18 = 30
30 - 18 = 12
18 - 12 = 6
12 - 6  = 6
6 - 6   = 0
```
**НОД** = 6

> На практике всегда используют алгоритм с делением, в силу того, что алгоритм с вычитанием медленный при больших числах.
